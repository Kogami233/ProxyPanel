### 系统要求

* PHP 7.3 +
* Mysql 5.5 +
* Memory 1G +
* Disk 10G +
* PHP Extensions: ZIP、XML、CURL、GD2、FileInfo、OpenSSL、Mbstring、PDO、Tokenizer、Ctype、JSON、BCMath
* Laravel 5.8

### 安装

{% hint style="info" %}
本项目基于 Laravel 开发，新手建议查看 laravel 的环境要求再进行部署。
